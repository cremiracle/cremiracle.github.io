C++随机数实现方法

C++中，库中包含rand()函数可以实现随机数，返回的随机数值的范围在 0 至 RAND_MAX 间。
如果要限制范围，具体用法为rand()% (b-a+1)+a；
但函数返回的数字实际上是用算法生成的，实际上并不是随机的。它是根据种子生成的，根据不同的种子产生不同的随机序列。系统默认的种子在系统开机时就已经确认，所以说每次使用的随机序列都是固定的。例如：

void srand(unsigned int seed);
srand()函数就是用来设置rand()函数的种子的。根据不同的输入参数可以产生不同的种子。通常使用time函数作为srand函数的输入参数。time()函数包含在time.h库中。
time函数会返回1970年1月1日至今所经历的时间（以秒为单位）。
